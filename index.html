HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Head elements contain various metadata for web applications such as: title, style, base, link, meta, script, charset
  • Any code is closed off with a / before the applicable tag i.e. /head

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • cards establish class selectors allowing one to apply style or format to specific items
  • * is a wild card selector which applies the indicated format to the entire HTML

Git

  • git status: checks what branch we are currently on
  • git branch: shows complete list of available branches with * indicating active branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git flow: git add -A > git commit -m "your message here..." > git pull origin main > git push origin name of branch
  • Git Command Cheat Sheet

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Handles the logic on a web application; allows users to interact with it
  • Use separate script.cs file via link in HTML code to incorporate all JS code
  • arrays allow for the processing of a large list of if statements via for loops
